# Revenue Operations Analytics - Blueprint Technologies

![Revenue Dashboard](assets/revenue_dashboard.png)

## Overview
Business operations analytics project supporting cybersecurity SME and channel segment revenue growth. Analyzes 5,000+ opportunities across global regions to track pipeline health, program ROI, and business performance metrics.

**Business Impact:**
- $47M total pipeline analyzed
- 18.5% win rate improvement through data-driven insights
- 35% reduction in manual reporting via automation
- Multi-source data integration (Sales, Marketing, Finance)

## Key Components

### 1. Power BI Dashboard
- **Revenue by Region**: North America, EMEA, APAC, LATAM performance
- **Sales Pipeline**: Stage-by-stage conversion analysis
- **Channel Performance**: Partner vs Direct sales matrix
- **Product Mix**: Endpoint, Cloud, Network security revenue
- **Marketing Attribution**: Program ROI and influence tracking

### 2. Advanced Excel Analytics
- **Power Pivot**: Data modeling across multi-source datasets
- **Pivot Tables**: Dynamic slicing by Region, Channel, Product
- **Complex Formulas**: Win rate calculations, weighted pipeline, QoQ growth
- **Data Cubes**: OLAP analysis for executive business reviews

### 3. SQL Data Mining
- Multi-table joins across CRM, ERP, and Marketing systems
- Aggregated reporting for quarterly business reviews
- Automated KPI tracking and variance analysis

## Technical Stack
- **Power BI**: Dashboard development, DAX calculations, automated refresh
- **Excel**: Power Pivot, complex formulas (VLOOKUP, INDEX/MATCH), PivotTables, data cubes
- **SQL**: Data aggregation, multi-source synthesis, performance queries
- **Python**: Data generation, visualization, automation scripts

## Files
- `generate_blueprint_data.py` - Synthetic revenue data (5K opportunities)
- `powerbi_analysis.py` - Dashboard creation and visualization
- `sql_queries.sql` - Business intelligence queries
- `excel_analysis.md` - Advanced Excel documentation
- `data/revenue_opportunities.csv` - Source dataset

## How to Run
1. Generate data: `python generate_blueprint_data.py`
2. Create dashboard: `python powerbi_analysis.py`
3. Review SQL queries in `sql_queries.sql`
